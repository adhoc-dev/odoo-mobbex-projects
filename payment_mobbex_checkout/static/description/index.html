Â¿Como usar?
1. Ir a Facturacion / Contabilidad

En el menu principal ingresaremos a la opcion Facturacion / Contabilidad

2. Seleccionar Metodos de Pagos

Primero desplegamos el menu de Configuracion. Y luego seleccionamos la opcion de "Metodos de Pago"

3. Activar Mobbex

Buscamos el metodo de pago Mobbex y le damos click al boton "Activar".

4. Habilitar y configurar Mobbex

Vamos a tener que seleccionar "Habilitado" en "Proveedor Estado". Y configurar la clave api y token de acceso en base a los datos obtenidos de la consola de Mobbex. Y por ultimo le damos al boton guardar.
